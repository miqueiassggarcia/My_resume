Criação de ambientes de estudo e testes para deploy de aplicações backend e frontend em cloud, utilizando serviços da AWS e boas práticas de infraestrutura e automação.

O projeto teve como objetivo consolidar conhecimentos em Linux, Docker e AWS, simulando ambientes reais de produção com foco em organização, segurança e repetibilidade.

  • Deploy de aplicações em EC2
  • Armazenamento e assets com S3
  • Gerenciamento de permissões com IAM
  • Automação de setup com scripts Bash
  • Versionamento e organização de ambientes Docker
