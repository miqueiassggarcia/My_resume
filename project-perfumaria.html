Sistema de Gerenciamento de Estoque para Perfumaria

  • Live Demo
  • Código Fonte

Desenvolvimento completo de um sistema de gerenciamento de estoque para uma perfumaria, com foco em controle de produtos, categorias, entradas e saídas de mercadorias, além de controle de usuários e permissões.

O backend foi desenvolvido em Java com Spring Boot, expondo APIs REST seguras e organizadas, enquanto o frontend foi construído em Angular, garantindo uma interface responsiva e intuitiva para uso diário.

  • Backend em Java + Spring Boot seguindo SOLID, Clean Code e DRY
  • Banco de dados Postgres com modelagem relacional e consultas otimizadas
  • Autenticação e autorização com Keycloak (OIDC)
  • Docker para padronização e portabilidade do ambiente
  • Deploy experimental em AWS (EC2, S3, IAM)
